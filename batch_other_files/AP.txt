2. Enable Auto-Suggestions As You Type (Jupyter Notebook 7+) 
If you are using the modern Jupyter Notebook v7 or greater, you can enable continuous code completion directly in the settings without downloading extensions: 

1. Click Settings in the top menu bar. 
2. Select Settings Editor from the dropdown. 
3. Click on Code Completion in the left sidebar. 
4. Check the box for Enable Autocompletion. [4]
